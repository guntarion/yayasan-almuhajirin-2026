  2. API Endpoints

  | Endpoint                | Method         | Description                    |
  |-------------------------|----------------|--------------------------------|
  | /api/blog/articles      | GET            | List articles (with filters)   |
  | /api/blog/articles      | POST           | Create article (auth required) |
  | /api/blog/articles/[id] | GET/PUT/DELETE | Single article operations      |
  | /api/blog/categories    | GET/POST       | List/create categories         |
  | /api/blog/tags          | GET/POST       | List/create tags               |
  | /api/blog/units         | GET            | List all units                 |

  3. Before Testing - Database Setup Required

  You need to run the database migration and seed first:

  # Apply schema changes to database
  npx prisma db push

  # Seed units and categories
  npm run db:seed

  # Optional: View data in Prisma Studio
  npx prisma studio

  4. Create Test Article via API

  curl -X POST http://localhost:3000/api/blog/articles \
    -H "Content-Type: application/json" \
    -d '{
      "title": "Test Artikel Pertama",
      "content": "<p>Ini adalah konten artikel...</p>",
      "categoryId": "<category-id-from-db>",
      "unitId": "masjid",
      "status": "PUBLISHED"
    }'
